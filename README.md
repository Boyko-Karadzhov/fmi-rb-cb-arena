# fmi-rb-cb-arena
Project for Ruby course at FMI. A multiplayer guessing game.

## Prerequisites

- Ruby > 2
- Bundler
- Node.js

## Installation

1. Clone the repo
2. `bundler install`
3. `rake install`

## Run

`rake start`

A web client is accessible at http://localhost:7026/

## Test

`rake spec` - runs tests
`rake` - runs tests and style analysis
